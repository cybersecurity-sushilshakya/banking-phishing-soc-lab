# Banking Phishing SOC Lab

This project demonstrates a phishing attack simulation and SOC investigation environment.

## Lab Objectives

- Simulate phishing email attack
- Capture victim credentials
- Investigate attack using IDS and packet analysis

## Tools Used

- Parrot OS
- FakeSMTP mail server
- Social Engineering Toolkit (SET)
- Phishing website
- Thunderbird email client
- Snort IDS
- Wireshark packet analysis
